# 행렬

## 왜 행렬인가?
연립방정식을 행렬 방식으로 표기하면 간편함

### 기계학습에서 행렬의 이점
- 간결한 표기
- 실수 감소
- 컴퓨터에 적합한 방식


### 용어의 정리
- 스칼라: 크기
  - 질량, 부피, 거리, 속력, 실수, 정수

- 백터: 크기와 방향
  - 속도, 가속도, 운동량, 전기장

- 행렬: 2차원 배열
- 텐서: N차원 배열


행렬의 내적: X를 T한후 Y랑 곱 >> 결과값이 스칼라
행렬의 외적: Y를 T한 후, X랑 곱 >> 결과값이 행렬

전치(T): 행렬 내의 원소를 대각선 기준으로 위치를 바꾸는것
